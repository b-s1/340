This is a sample CS340 Project website based on the BSG Database.

Menu of operations

  • List, Add, Update and Delete Persons (perform Create, Read, Update, Delete operations on a single entity) and filter by Homeland and search by First name
  • Change Homeland of a Person (change 1-to-M relationships)
  • List and add new planets
  • List and add new certifications
  • List and add new spaceships
  • Filter people by Certifications they have
